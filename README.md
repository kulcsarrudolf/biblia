# Biblia (RÚF)

## ❓ How to use?

```bash
$ node index --p="Zsolt 100"
```

```bash
$ node index --p="Zsolt 100:1"
```

```bash
$ node index --p="Zsolt 100:1-2"
```

```bash
$ node index --showBooks
```
