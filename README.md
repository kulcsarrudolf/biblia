# Biblia (RÚF)

## CLI

### ❓ How to install?

```bash
$ npm i -g biblia-ruf
```

### ❓ How to use?

The list with commands:

```bash
$ biblia --p="Zsolt 100"
```

```bash
$ biblia --p="Zsolt 100:1"
```

```bash
$ biblia --p="Zsolt 100:1-2"
```

```bash
$ biblia --showBooks
```

```bash
$ biblia --showBooks --old
```

```bash
$ biblia --showBooks --new
```

```bash
$ biblia --bookDetails Zsolt
```
